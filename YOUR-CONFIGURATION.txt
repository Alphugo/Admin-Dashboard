HOW TO GET GMAIL APP PASSWORD:
==========================================

Step 1: Enable 2-Factor Authentication
----------------------------------------
1. Go to: https://myaccount.google.com
2. Click "Security" (left sidebar)
3. Click "2-Step Verification"
4. Follow prompts to enable it

Step 2: Generate App Password
--------------------------------
1. In Security settings, find "App passwords"
2. Click "App passwords"
3. Select "Mail"
4. Select "Other (Custom name)"
5. Enter "Kina Resort" 
6. Click "Generate"
7. Copy the 16-character password (example: abcd efgh ijkl mnop)

Step 3: Update Server Code
--------------------------
Edit server.js around line 44, replace:

OLD: pass: process.env.SMTP_PASSWORD || 'password123'
NEW: pass: process.env.SMTP_PASSWORD || 'YOUR-APP-PASSWORD-HERE'

Example:
pass: process.env.SMTP_PASSWORD || 'abcd efgh ijkl mnop'

Or create a .env file with:
SMTP_PASSWORD=abcd efgh ijkl mnop
